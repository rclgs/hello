# Hello, K8S!!!

An exercise on deploying a web application in Kubernetes using the `minikube` distribution on my local machine.

## Background

I'm looking to boost my Kubernetes skills, so I'm going to try deploying a simple Django app on minikube and see if I can access it through a browser.

## Understanding

* First I am going to create a Django project with `uv`
* Then I will create a Dockerfile for the Django project
* Then I will create a Docker Compose file for local development
* Finally try and deploy it to `minikube`
